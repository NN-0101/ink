## 为什么叫 ink？

墨水，书写最本质的载体。没有多余的花哨，回归文字本身。四个字母，简洁如它的名字。

## 特性

- 📝 **多标签页编辑** — 支持同时打开多个文件，标签页可拖动排序
- 🌳 **树形文件列表** — 按目录结构组织打开的文件，支持折叠/展开/定位
- 🔍 **强大的查找替换** — 支持正则表达式、区分大小写、全词匹配、高亮所有匹配
- 🎨 **多语言语法高亮** — 支持 20+ 种编程语言
- 📏 **行号显示** — 清晰的左侧行号区域，带有视觉分隔线
- 📐 **缩进参考线** — 智能显示代码块的缩进层级（支持空格和 Tab）
- 👁️ **Markdown 预览** — 实时预览 Markdown 文件渲染效果
- ⚡ **高效快捷键** — `Ctrl+D` 删除整行，`Ctrl+U` 大小写转换
- ⌨️ **全局快捷键** — `Ctrl+F/R` 全局查找替换，`F3/Shift+F3` 查找下一个/上一个
- 💾 **会话管理** — 自动保存和恢复最近打开的文件列表
- 🖱️ **右键上下文菜单** — 文件树支持打开文件位置、CMD/Git Bash 终端
- 🎯 **智能缩进参考线** — 根据文件类型自动决定是否显示
